第三章 Object Literal

Literal的好处

  • 使用字面量能节省代码
  • 使用constructor会给人js有class的错觉
  • 当给Object()提供参数时，Object可能会根据参数类型讲构造委派给其他函数

使用new调用函数的不同

使用new调用函数时，编译器自动为我们创建了一个对象，这个对象的prototype指向Person.prototype，并且将这个对象赋给了this。在函数的最后，若本来没有返回值，this会自动作为返回值。
当我们确认一个函数的作用是构造对象时，我们应该使这个函数不论是否用new调用，都可以返回正确的对象，可以像下面这样做。

第四章 Function

函数的特点

  • 函数是第一等公民
    • 可以在运行时动态生成（new Funciton()）
    • 可以赋值给变量
    • 可以作为函数参数，也可作为函数返回值
  • 函数是对象
    • 可以为其添加属性
  • 函数提供作用域
    • 任何在函数内部通过var定义的变量都是局部变量

函数的三种形式

  • 命名函数表达式
  • 匿名函数表达式
  • 函数声明

区别在于匿名函数表达式没有name属性，其余两者有。除此之外，函数声明会提升到作用域的最顶部。
