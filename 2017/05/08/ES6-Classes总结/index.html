Class特点

  1. class声明不会提升，像let声明一样。
  2. class里面的代码自动运行在严格模式下
  3. 所有的方法默认不可遍历
  4. 所有的方法都不能用new调用
  5. 必须用new调用class
  6. 不能在class方法里面给class名赋值

子类如果有constructor函数，必须使用super，如果没有使用constructor，则会自动产生一个，并且在内部自动调用super。
super函数有以下特点:

  1. 只能在子类的constructor内使用super
  2. 必须在给this添加属性前使用super，因为super负责初始化this
  3. 避免使用super的唯一办法是在constructor内返回一个对象
