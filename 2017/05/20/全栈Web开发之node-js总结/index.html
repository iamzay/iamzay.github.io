Intro to MongoDB

安装好MongoDB后进入某文件夹下，然后执行mongod --dbpath=data，这样会将当前目录下的data文件夹作为数据库文件夹并且启动server。之后在另外的控制台输入mongo就连接到了server。

Mongoose ODN

Mongoose是另一个提供了更多特性的node module。Mongoose是基于MongoDB node module的，所以后者的特性在Mongoose中都可使用。Mongoose能控制document结构。

Basic Authentication

利用http请求报文的Authorization这个header属性进行验证，提供账号和密码。若没有提供该属性或者账户或密码错误，响应报文中会包含WWW-Authenticate属性要求客户端继续验证。

Cookies and Express Sessions

服务器将cookie发送给客户端，此后客户端在发送请求时都加上这个cookie，这样可以避免重复验证，服务器依然能识别出客户端。
express session在发送cookies时加上session id，这个id可以用来在server side查询用户的信息。这个信息保存在内存中，所以server重启后就没了。
