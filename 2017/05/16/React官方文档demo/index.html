React的思想:将组件分成不同的层次，确定状态是什么，将状态作为属性从高层的组件传给低层的组件。同时也将更改状态的函数往下传递，在低层次的组件中更改高层组件的状态，从而导致重新渲染。关键在于确定状态是什么（像下面的例子中就是用户的输入）和哪个组件拥有状态（其他组件只拥有属性）。
