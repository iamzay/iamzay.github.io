有关Promise.resolve()

发表于 2017-08-02 | 分类于 Web ， Javascript |

Promise.resolve()是一种构造Promise的方式，它接受三种不同的参数，分别有不同的行为。

  1. Promise
    当传递一个Promise作为参数时，函数将其原样返回。
  2. non-promise,non-thenable
    当传递一个普通值时，函数返回一个处于fulfilled状态的promise，且值为这个普通值。
  3. non-promise thenable
    当传递一个thenable作为参数时，Promise会使用then属性代表的函数构造一个promise返回：

上面resolve返回的是一个处于rejected状态的promise。由此可见，Promise.resolve并非总是返回fulfilled promise。但与之对应的Promise.reject函数总是返回rejected promise。

Promise构造函数接受的回调函数接受两个参数，resovle和reject。reject会使promise进入rejected，resovle的效果和Promise.resolve函数类似，可以使promise进入fulfilled或rejected。
